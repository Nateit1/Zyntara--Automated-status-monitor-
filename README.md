# 🔴 Zyntara - Automated Status Monitor

> Add any URL and Zyntara checks if it's online or offline automatically — with a live progress bar and status indicators.

## 📌 What It Does

Zyntara lets you monitor multiple websites at once. Add any URL and it automatically pings it at your chosen interval — showing green if it's online, red if it's offline, and when it was last checked.

## 🚀 Features

- Add and monitor multiple URLs simultaneously
- Automated checks every 30s, 1m, or 5m — your choice
- Live progress bar showing time until next check
- Green/red pulsing status dots for online/offline
- Timestamps showing when each site was last checked
- Sites persist across sessions via localStorage
- Press Enter or click Add to add a URL

## 🏃 How to Run

No installation needed — runs entirely in the browser.

```bash
# Clone the repo
git clone https://github.com/Nateit1/zyntara.git
cd zyntara

# Open in browser
open index.html
```
